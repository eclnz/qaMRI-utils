# MRI Utility Functions

## Requirements

- Python + pip

## Installation

1. Clone the repository:

```bash
git clone __
cd repo
```

2. Create virtual env:

```bash
python3 -m venv venv
```

3. Activate virtual env:

```bash 
source venv/bin/activate # mac
```

4. Call functionality from the command line
```bash
python3 dcm2bids raw_data_dir bids_dir
```
